**Flex** (от англ. flexible box) — это модель компоновки в CSS, предназначенная для создания гибких и адаптивных макетов веб-страниц. Эта технология позволяет легко управлять расположением и выравниванием элементов внутри контейнера, даже если их размеры динамически изменяются.

### Основные принципы работы

Flexbox работает на основе **контейнера** и его **дочерних элементов**. После присвоения контейнеру свойства `display: flex`, все элементы внутри него становятся flex-элементами и подчиняются специальным правилам позиционирования.

### Ключевые оси в Flexbox

В системе Flexbox используются две основные оси:

- **Главная ось** (main axis) — определяет направление расположения элементов (по умолчанию слева направо)
- **Поперечная ось** (cross axis) — перпендикулярна главной оси

### Основные свойства Flexbox

- **display: flex** — превращает элемент в flex-контейнер
- **flex-direction** — задаёт направление расположения элементов (row, row-reverse, column, column-reverse)
- **justify-content** — управляет выравниванием элементов вдоль главной оси
- **align-items** — отвечает за выравнивание элементов вдоль поперечной оси
- **flex-wrap** — определяет поведение элементов при нехватке места (перенос на новую строку)

### Практическое применение

Flexbox особенно полезен в следующих случаях:

- Создание адаптивных макетов
- Центрирование элементов
- Автоматическое распределение пространства
- Выравнивание элементов по высоте
- Создание гибких навигационных меню
- Разработка отзывчивых карточек и галерей

### Пример использования

```
.container {
    display: flex;
    flex-direction: row;
    justify-content: space-between;
    align-items: center;
}

.item {
    background-color: lightblue;
    padding: 10px;
    margin: 5px;
}
<div class="container">
    <div class="item">Элемент 1</div>
    <div class="item">Элемент 2</div>
    <div class="item">Элемент 3</div>
</div>
```

### Преимущества Flexbox

- Простота выравнивания элементов
- Гибкость в управлении пространством
- Адаптивность к разным размерам экрана
- Возможность изменения порядка элементов
- Эффективное использование пространства

Flexbox стал стандартом современной веб-разработки благодаря своей мощности и гибкости в создании сложных макетов без использования сложных расчётов и дополнительных обёрток.
